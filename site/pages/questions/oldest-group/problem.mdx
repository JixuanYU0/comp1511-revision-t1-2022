# Oldest Group

There a  groups of people with each group  containing  people.

Find the group with the highest sum of ages.

If two groups have the same sum of ages, print the group with the lower group number.

**Input Format**

The first line of input contains the integer , the number of groups.

The following  pairs of lines each describe the ages of people in groups labelled  to  in order.

The first line of each pair contains an integer  for the number of people in group . The second line contains  numbers, the ages of each person in group .

**Constraints**

For all test cases:
- $1\le N \le 1000$
- $1\le K_i \le 1000$
- The age of any person $A$ is an integer $1\le A \le 100$

**Output Format**

Output an integer $i$ for the group with the largest age sum.

**Sample Input 0**

```
2
5
26 19 31 23 28
4
56 34 47 29
```

**Sample Output 0**

```
2
```

**Sample Input 1**

```
3
2
35 27
2 48 23
1
98
```


**Sample Output 1**

```
3
```
